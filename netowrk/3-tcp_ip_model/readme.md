# Internet Route Mapping Script

- Thiago Messias Santos
- São Paulo, 20/05/2026

## Description

This script uses `tracepath` to show the route between your machine and a remote server.

Each step (hop) represents a router along the path.

## What it shows

- Route (path) to the target server  
- Number of hops (routers)  
- Latency between each hop
