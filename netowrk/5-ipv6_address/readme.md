# 🌐 IPv6 Analysis Script

- Thiago Messias Santos
- São Paulo, 20/05/2026

## Description

This script demonstrates basic IPv6 usage on Linux.

It displays IPv6 addresses and tests both local and external connectivity.

## What it does

- Shows IPv6 addresses on the system  
- Tests local connectivity using loopback (::1)  
- Tests external IPv6 connectivity  
- Provides a tip for using IPv6 in browsers
