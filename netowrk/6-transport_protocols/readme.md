# TCP vs UDP Script

- Thiago Messias Santos
- São Paulo, 20/05/2026

## Description

This script demonstrates the difference between TCP and UDP protocols using local connections.

It creates simple servers and sends messages between them using netcat.

## What it does

- Creates a TCP server and sends a message  
- Creates a UDP server and sends a message  
- Displays received messages  
- Highlights differences between TCP and UDP
