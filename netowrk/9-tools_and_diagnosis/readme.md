# Network Troubleshooting Script

- Thiago Messias Santos
- São Paulo, 20/05/2026

## Description

This script performs a basic network health check on a Linux system.

It verifies connectivity, DNS resolution, and active services.

## What it does

- Displays active network interfaces  
- Identifies the default gateway  
- Tests connectivity to the gateway  
- Tests internet access  
- Verifies DNS resolution  
- Lists listening ports and services
