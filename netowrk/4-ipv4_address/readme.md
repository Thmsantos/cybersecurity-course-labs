# IPv4 Address Configuration Script

- Thiago Messias Santos
- São Paulo, 20/05/2026

## Description

This script demonstrates how to manually configure an IPv4 address on a Linux interface.

It adds an IP address, verifies it, tests connectivity, and then removes it.

## What it does

- Adds a new IP address to an interface  
- Displays current IP configuration  
- Tests connectivity using ping  
- Removes the configured IP
